# Secure Login System

## Overview
A secure login and registration system built using Flask, SQLite, and bcrypt password hashing.

## Features
- User Registration
- User Login
- Password Hashing with bcrypt
- Input Validation
- SQL Injection Protection using Parameterized Queries
- Session Management
- Logout Functionality

## Installation

```bash
pip install -r requirements.txt
```

## Run Application

```bash
python app.py
```

Open:

http://127.0.0.1:5000
